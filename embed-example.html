Embed: Park City Olympics Tracker

Copy any of the code snippets below to embed the tracker on your site.

Option 1: Fixed height (simplest)

Set a fixed pixel height. Works everywhere. Adjust the height value to fit your layout.

<iframe
  src="https://utah2026.townlift.com/"
  style="width: 100%; height: 800px; border: none;"
  loading="lazy"
  title="Park City at the 2026 Winter Olympics"
></iframe>

Option 2: Viewport-height responsive (recommended)

Uses 100dvh so the embed fills the screen height on any device. Falls back to 100vh for older browsers.

<div style="width: 100%; height: 100vh; height: 100dvh; overflow: hidden;">
  <iframe
    src="https://utah2026.townlift.com/"
    style="width: 100%; height: 100%; border: none;"
    loading="lazy"
    title="Park City at the 2026 Winter Olympics"
  ></iframe>
</div>

Option 3: Auto-resizing with JavaScript

The iframe resizes itself to match the content height. No scrollbar inside the iframe. Best for embedding within article pages.

<div id="pc-olympics-embed" style="width: 100%;">
  <iframe
    id="pc-olympics-iframe"
    src="https://utah2026.townlift.com/"
    style="width: 100%; border: none; min-height: 600px;"
    loading="lazy"
    title="Park City at the 2026 Winter Olympics"
  ></iframe>
</div>
<script>
(function() {
  var iframe = document.getElementById('pc-olympics-iframe');
  window.addEventListener('message', function(e) {
    if (e.origin !== 'https://utah2026.townlift.com') return;
    if (e.data && e.data.type === 'pc-olympics-resize') {
      iframe.style.height = e.data.height + 'px';
    }
  });
})();
</script>
